Google Site Verification

Este arquivo é usado para verificar a propriedade do domínio performaxis.com.br no Google Search Console.

Após configurar o Google Search Console, substitua este arquivo pelo arquivo de verificação fornecido pelo Google.

Instruções:

  1. Acesse: Google Search Console
  2. Adicione a propriedade: performaxis.com.br
  3. Escolha o método de verificação: "Arquivo HTML"
  4. Baixe o arquivo de verificação fornecido pelo Google
  5. Substitua este arquivo pelo arquivo baixado
  6. Faça o deploy e clique em "Verificar" no Google Search Console

Após a Verificação:

  • Submeta o sitemap.xml: https://performaxis.com.br/sitemap.xml
  • Solicite indexação da página principal
  • Configure o domínio preferido (com ou sem www)
  • Monitore a indexação e cobertura nas próximas semanas
